## Set Up

Run `npm install`

## Deployment

`npm run deploy` and it will output addresses in versions.json. And the JSON ABI are in the artifects folder. 

## Test

`npm run test`

## Coverage

`npm run coverage`

## Lint

`npm run lint:sol`

## Prettier

`npm run prettier:sol`
